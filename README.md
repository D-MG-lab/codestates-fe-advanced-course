### 프로젝트 실행 방법
1. 필수 구현 사항 확인 후 Figma 활용하여 페이지 디자인
2. 필요한 스택 확인
3. '필수 구현 사항' 구현
4. 추가 구현 가능한 부분 구현

### 사용한 스택 목록
- **JavaScript**
- **React**
- **axios**
- **Styled-component**

### 구현한 기능 목록 (Software Requirement Specification)
- 게시물 리스트 구현
- 게시물 상세페이지 및 댓글 나열
- 게시물 리스트 페이지네이션 구현

### 구현 방법 혹은 구현하면서 어려웠던 점
- 처음 배포 할 때, 깃허브 페이지로 배포하려 했으나 네트워크 빌드 오류가 해결이 되지 않아 AWS S3 배포로 방식을 바꿨습니다.
